# Jogo de estratégia 

- Este projeto consiste em um jogo de estratégia desenvolvido durante o minicurso de Programação Orientada a Objetos, desenvolvido na linguagem C++. 

#  IDEIA GERAL
- O projeto será um jogo de estratégia em turnos. Nele, o jogador irá gerenciar uma
aldeia coletando recursos; construindo e aprimorando estruturas; e treinando unidades com
o objetivo de conquistar outras aldeias controladas pela máquina.
